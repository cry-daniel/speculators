# PAMS-Verify

Probability- and Memory-Aware Sparse Verification for speculative decoding.

This directory is a self-contained experiment harness for the requirements in
`../speclink.md`. It separates evidence labels clearly:

- `offline_simulation`: trace, acceptance, union-growth, and mask-planner work.
- `kernel_microbenchmark`: torch reference sparse-attention timing.
- `patched_vLLM_end_to_end`: required for GO, currently not achieved.

The current runnable path uses deterministic synthetic fallback traces when
online draft attention and dense target labels are unavailable. It does not use
dense target labels to select online masks.

## Quick Start

Run from this directory:

```bash
bash scripts/run_smoke.sh
```

Full registered workflow:

```bash
bash scripts/run_all.sh
```

The final decision is generated by:

```bash
python -m pams.report --root experiments --output reports/final_report.md
```

## Evidence Policy

Do not claim PAMS speedup unless `experiments/10_end2end` contains a live
patched-vLLM result. Offline and microbenchmark gains are useful diagnostics but
are not sufficient for GO.
